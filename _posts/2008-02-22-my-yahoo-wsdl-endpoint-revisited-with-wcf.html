In a "previous post":/2008/02/a-wsdl-file-for-yahoo-rest-api, I made an adapter in order to offer a WSDL endpoint to Yahoo! REST Api.

In this post, I will transform my solution in order to use "Windows Communication Foundation":http://netfx3.com/content/WCFHome.aspx to expose the endpoint (as WSDL again, but as you may know, it is easy to change the way endpoint are exposed with WCF.

h3. Windows Communication Foundation

WCF is based on "ABC":
* _Address_ : where the message can be sent
* _Binding_ : how to send the message (Tcp, Http, Named Pipes, MSMQ...)
* _Contract_ : what is exposed to the outside world
